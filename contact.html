General Information

Before you contact us with a question, please read the FAQ and the Documentation if your question might already be answered there.

Bugs and Feature Requests

YouScope is hosted on GitHub. As such, the easiest way to report bugs, feature requests, and similar is to create a so called "issue", which you can do here: https://github.com/langmo/youscope/issues

The GitHub "issue" system has the advantage that it is easier to track issues, as well as that other users can comment on these issues. Thus, we prefer issues over e-mails, even though you can still write us an e-mail if you prefer to do so.

Contact by E-Mail

We are interested in your user-experience with YouScope. If you are using YouScope and are happy with it – tell us! If you found a bug, miss a feature, or even stopped using YouScope for one or the other reason – then there is even more reason to tell us!
We constantly try to improve YouScope, but this will only lead to success if you tell us what you like, and what you don't.

To contact us, please write a mail to
info@youscope.org
